# Feed Restyle Notes (뉴닉 톤 + X 사용성)

## 현재 제안 (O → C/G 공유용)
- 피드 형태: 커뮤니티형 단일 피드(트위터 카드 크기), 한 화면에 1.2~1.5개 노출.
- 인터랙션: 카드 인라인 접힘/펼침 토글(한 번 탭/클릭 → 펼쳐짐, 위로 스크롤/버튼 → 접힘). 모달 사용 안 함.
- 카드 접힘 상태:
  - 제목 + 메타 뱃지(“오늘 n개 · 업데이트 hh:mm KST · 국가 n개”)
  - “왜 중요한가” 한 줄(80자 내, 키워드 볼드)
  - 미니 스펙트럼(3색 막대) + “매체 톤” 라벨
  - 썸네일/배경 이미지는 텍스트 없이 조미료만(오른쪽 상단/배경)
  - 행동: “한 번에 보기”(펼침), 하트/북마크(옵션)
- 카드 펼침 상태:
  - What happened / Why it matters 2~3문장(뉴닉 톤)
  - 국가별 미니 리스트(국기 + ▲/■/▼ + 한줄, 최대 4개, 나머지 “더 보기”)
  - 링크 모음(“더 읽기 3건”)
  - 닫기 토글(상단/하단 둘 다)
- 섹션:
  - 상단 메타바(고정): “오늘 n개 · 업데이트 hh:mm KST · G10+CN/RU”
  - 피드 내 섹션: Top 3 = “지금 바로 봐야 할 소식”, 나머지 = “잠시 후에 살펴볼 소식”
- 톤/카피:
  - Hero: “5분 안에 오늘의 세계를 살펴보세요.” / “핵심 이슈 3–5개, 각기 다른 시선까지 한눈에 담았습니다.”
  - 스펙트럼 캡션: “이 막대는 매체 톤을 보여줘요. 정답을 말하는 건 아니에요.” / “데이터: Supabase topics · 업데이트: hh:mm KST”
  - 상태: 로딩 “데이터를 준비하고 있어요. 잠시만요.” / 에러 “조금 늦어지고 있어요. 새로고침하거나 잠시 후 다시 시도해 주세요.” / 빈 상태 “23:30 KST 이후에 다시 열어볼게요.”
- 이미지 사용 원칙: 텍스트를 이미지 위에 얹지 않음. 배경/썸네일로 분위기만. 텍스트는 네이티브로.

## C에게 요청할 작업
1) topics API 확장: `meta.count`(오늘 토픽 수), `meta.updatedAt`(max(updated_at) 또는 date 기반) 반환. Hero/메타바/캡션에 바인딩.
2) 리스트 레이아웃 변경: 인라인 접힘/펼침 토글 추가, 카드 패딩 축소, 섹션 헤더(Top 3 vs 추가) 추가.
3) 카피 적용: Hero/메타/카드 Why-line/상태 메시지/스펙트럼 캡션을 제안된 톤으로 교체.
4) 미니 스펙트럼: 라벨을 “매체 톤” 등 직관 문구로 노출, 툴팁 없이도 이해되게 기본 캡션 표시.

## G에게 요청할 협업 포인트
- 카피 미세 조정: What/Why 2~3문장 예시, 친절 톤 유지.
- 이미지 톤: 텍스트 없는 조미료 이미지 가이드(색/질감/패턴) 제안.
- 모바일 QA: 한 화면 정보량/패딩 최적화, 터치 목표 44px 유지.

## 추가 참고
- 기존 Newneek 톤+X 사용성을 결합: 빠른 스크롤 + 단일 탭으로 확장/복귀.
- 지향: “일반 피드처럼 부담 없이 훑되, 펼치면 총체적 느낌”을 제공.
