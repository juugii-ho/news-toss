## 🚨 발견된 문제점 (우선순위별)

### CRITICAL (즉시 개선 필요)

#### 1. 모바일 경험 최악
**문제:**
- Country Breakdown 2-column 너무 좁음
- Spectrum bar 터치하기 힘듦 (작음)
- Hover tooltip 모바일 작동 안 함

**영향:** 70% 사용자 (모바일) 경험 저하
**해결책:**
- 모바일: 1-column layout
- Spectrum bar 높이 16px → 20px
- Tap 시 explanation modal

#### 2. 핵심 컨텐츠 부재
**문제:**
- Topic Summary = 자동 생성 (무의미)
- Country Summary = null
- "왜 중요한가" 답 없음

**영향:** 사용자 가치 제공 실패
**해결책:**
- G에게 Summary 생성 요청 (최우선)
- 임시: "분석 준비 중" 표시
- 또는: 관련 기사 제목 표시

#### 3. 맥락 부족
**문제:**
- Spectrum 의미 모름 (특히 모바일)
- 국가 선정 기준 불명확
- 데이터 신선도 불명확

**영향:** 신뢰도 저하, 이탈 증가
**해결책:**
- 모바일용 "?" 버튼
- Footer "Data Sources" 링크
- Header "Last Updated: 2h ago"

---

### HIGH (중요)

#### 4. 첫 인상 약함
**문제:** Hero 평범, 스크롤 유도 부족, 가치 제안 모호
**해결책:** Hero 카피 개선, 스크롤 인디케이터, 업데이트 뱃지

#### 5. 카드 정보 부족
**문제:** 날짜 없음, 전체 수 모름, 우선순위 불명확
**해결책:** 각 카드 날짜, 상단에 "오늘의 토픽 5개", Hot 뱃지

#### 6. 네비게이션 불편
**문제:** Detail→Back 원래 위치 안 돌아감, 카드 간 이동 번거로움
**해결책:** Scroll position 복원, "다음 토픽" 버튼

---

### MEDIUM (개선 권장)

#### 7. 모바일 터치 타겟
**문제:** 버튼 작음 (44px 미만), 링크 간격 좁음
**해결책:** 모든 터치 요소 44px 이상, 카드 전체 clickable

#### 8. 로딩 경험
**문제:** Skeleton 길어지면 답답, 진행 상황 모름
**해결책:** 빠른 네트워크 skeleton skip, Progress indicator

#### 9. 읽기 경험
**문제:** 모바일 텍스트 작음, Line height 좁음
**해결책:** Font 16px → 18px, Line height 1.5 → 1.6
