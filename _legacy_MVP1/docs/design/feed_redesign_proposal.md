# Feed 재설계 제안

## 현재 문제
- 2-column grid → 너무 정적, 데스크톱 중심
- 카드가 너무 무거움 (Country Breakdown, 왜 중요한가 등)
- "기능 전시형" → 사용자가 압도됨

## 목표
**X.com + 뉴닉 사이의 사용성**
- 가볍게 스크롤하면서 소비
- 한 번의 터치/클릭으로 상세 확인
- 뉴닉스러운 친근한 문법
- 이미지 중심, 간결함

---

## 새로운 피드 디자인

### 1. Feed Item (축약 카드)

```
┌─────────────────────────────────┐
│ 🇰🇷 트럼프, 관세 폭탄 예고       │ ← 제목만 (큼직하게)
│ Trump Announces Tariff Plan     │ ← 영문 (작게)
│                                  │
│ 🟢🟢🟢⚪⚪⚪🟡🟡                  │ ← 스펙트럼 (한 줄, 비주얼)
│                                  │
│ 💬 127개 기사 · 🌍 12개국       │ ← 메타 (간결)
│ 🕐 2시간 전                     │
└─────────────────────────────────┘
```

**특징:**
- Single column (모바일 우선)
- 높이 120-150px (컴팩트)
- 스펙트럼은 시각적 요소 (설명 X)
- 클릭하면 drawer/modal 열림

### 2. Detail Drawer (클릭 시)

```
┌─────────────────────────────────┐
│ [X 닫기]                        │
│                                  │
│ 🇰🇷 트럼프, 관세 폭탄 예고       │
│                                  │
│ 💡 왜 중요한가요?               │
│ 미국의 새로운 관세 정책이...     │
│                                  │
│ 🌍 국가별 반응                  │
│ 🇺🇸 United States               │
│ 🟢🟢🟢⚪⚪🟡                    │
│                                  │
│ 🇨🇳 China                       │
│ 🟡🟡🟡🟡⚪⚪                    │
│                                  │
│ 📰 관련 기사 127개              │
│ [기사 목록...]                  │
└─────────────────────────────────┘
```

**특징:**
- Bottom sheet (모바일) / Side panel (데스크톱)
- 스크롤 가능
- 닫기는 간단히 (X, 스와이프 다운)


## 구현 우선순위

### Phase 1: 피드 단순화 (1-2시간)
- [ ] Single column feed
- [ ] 컴팩트 카드 (제목, 스펙트럼, 메타)
- [ ] Country Breakdown 제거 (메인에서)
- [ ] "왜 중요한가" 제거 (메인에서)

### Phase 2: Detail Drawer (2-3시간)
- [ ] Bottom sheet / Side panel 구현
- [ ] 카드 클릭 시 drawer 열기
- [ ] 상세 내용 모두 drawer로 이동

### Phase 3: 톤 변경 (30분)
- [ ] "~해요" 체로 변경
- [ ] 아이콘 추가 (💡, 🌍, 📰)
- [ ] 친근한 문구

## 레퍼런스

**X.com 피드:**
- Single column
- 빠른 스크롤
- 탭하면 상세

**뉴닉:**
- 친근한 톤 ("~해요", "~군요")
- 이모지 사용
- 간결한 요약

**우리:**

- 스펙트럼 시각화 (우리만의 조미료)
- 한 번 탭으로 깊이 있는 정보
