# Persona Insight: Mobile Busy Power User

- Persona: 시간은 부족하지만 관심 토픽엔 깊게 몰입하는 깐깐한 모바일 사용자(뉴스·지정학 관심, 탐색 의지 있으나 초기 이탈 위험 높음).

## 관찰/반응
- 첫 화면에서 “무엇을 몇 개/얼마나 최신?”이 바로 보이지 않아 기대치 설정이 약함.
- 카드별 핵심 한줄(왜 중요한지)이 바로 안 보여 스펙트럼 전 단계에서 이탈 가능.
- 스펙트럼은 흥미롭지만 클릭 전에 추가 정보를 예고하지 않아 상세 보기 동기가 약함.
- 신뢰도: 데이터 수집 시각/소스 표기가 부족해 “얼마나 최신/어디서 왔는가”가 불명확.
- 모바일에서 카드 여백과 텍스트 비중이 커서 한 화면에 보이는 정보량이 적음.

## 개선 포인트 (우선순위 순)
1) Hero 바로 아래 메타 바: “오늘 n개 토픽 · 업데이트 23:30 KST · G10+CN/RU” 등 기대치 설정.
2) 카드 상단 “왜 중요한가” 한줄(≈80자) → 제목 바로 아래 배치.
3) 읽기 부담/범위 뱃지: “읽기 20초”, “국가 5개” 등과 접기/펼치기 토글을 더 명확하게.
4) 신뢰도 표기: 소스(주요 신문/RSS)와 마지막 갱신 시각을 헤더/빈 상태/카드 상단에 작은 텍스트로 노출.
5) 액션 예고: “View Full Analysis” 옆에 “국가별 톤 요약 + 긴 설명 제공” 등 링크가 열어줄 가치 명시.
6) 모바일 정보량 증가: 카드 패딩 소폭 축소, 제목 대비 강화로 한 화면에 더 많은 카드/정보 노출.
7) 몰입 유도 링크: 카드 하단에 “비슷한 토픽”/“다른 국가 관점 더 보기” 한줄 링크로 연쇄 탐색 촉진.
