# Topic Flow Concept (Typhoon/Cloud Metaphor)

## Idea
- 토픽을 태풍/구름처럼 표현해 생성→이동(변화)→분화/합류→소멸 흐름을 시각화.
- 신뢰도/투명성을 높이기 위한 “토픽 생태계” 애니메이션: 하루 단위 프레임 재생, 버블/입자 이동.

## Data Mapping
- 위치: 임베딩(제목/요약/기사 평균)을 2D(Umap/T-SNE)로 투영. 국가 가중치 적용 시 국가 분포 중심으로 약간 이동.
- 크기: 기사 수/참여 국가 수.
- 색: stance 평균/분산(예: 비판→파랑, 중립→회색, 호의→빨강), 갈등도 높으면 채도↑.
- 투명도/꼬리: 오래된 토픽 희미, 최근/활성 토픽 선명. 이동 경로에 꼬리/흔적 옵션.

## States & Links
- 매칭(다대다): 전일 vs 금일 유사도(thr)로 연결. 분화(A→B1,B2), 합류(A1,A2→B), 신규(연결 없음), 소멸(후속 없음).
- 변이: 유사도는 높지만 stance/국가 분포 변화량이 큰 경우 “양상 변화” 라벨.

## UX/Controls
- 재생/일시정지/타임슬라이더(날짜 점프).
- 툴팁: 제목, 국가수, 주요 키워드, stance 요약.
- 필터: 지역/주제/갈등도(stance 분산) 필터.
- 알림: “신규 폭풍”(신규 토픽), “분화/합류” 배지.
- 접근성: 색맹 대비 유지, 모바일 모드에선 단순 버블/라인 모드.

## Technical Notes
- 프리컴퓨트: 일자별 위치/링크 JSON 생성(서버), 클라이언트는 재생만.
- 성능: WebGL/Canvas(Pixi/Deck.gl 등), 모바일 폴백은 단순 SVG 점/라인.
- MVP: 2D 버블 + 날짜별 프레임 + 분화/합류/소멸 라벨 → 반응 좋으면 꼬리/입자 효과 추가.

## Next Steps (실행)
- 일별 토픽 임베딩/요약 저장 + 전후 유사도 기반 연결 JSON 생성.
- 간단 프리뷰: Plotly/Canvas로 날짜별 프레임 애니메이션(TSNE/UMAP 투영) 제작.
- 임계값/변화 라벨 튜닝: 분화/합류/신규/소멸 규칙을 테스트 데이터로 검증.
