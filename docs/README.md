# MVP2 프로젝트

이 프로젝트는 [프로젝트의 한 줄 설명]을 목표로 합니다.

## 빠른 시작

1.  이 리포지토리를 클론합니다.
2.  환경 변수를 설정합니다. (`.env.example` 참고)
3.  `make setup`을 실행하여 의존성을 설치합니다.
4.  `make dev`를 실행하여 개발 서버를 시작합니다.
5.  로그·결과물은 `outputs/` 하위에 보관하세요(gitignore 처리됨).

## 핵심 문서

-   **[VISION.md](./VISION.md)**: 제품의 비전, 목표, MVP 범위 및 성공 지표를 정의합니다.
-   **[RULES.md](./RULES.md)**: 우리(AI 에이전트)의 역할, 책임, 협업 규칙을 정의합니다.
-   **[WORK.md](./WORK.md)**: 현재 진행 중인 작업과 완료된 작업의 이력을 관리합니다.
-   **[DECISIONS.md](./DECISIONS.md)**: 주요 기술 및 정책 관련 의사결정 내역을 기록합니다.
-   **[ARCHITECTURE.md](./ARCHITECTURE.md)**: 시스템 아키텍처 다이어그램과 설명을 포함합니다.
-   **[DEPLOYMENT.md](./DEPLOYMENT.md)**: 배포 순서, 롤백 방법, 트러블슈팅 가이드를 제공합니다.
