# Production Secrets

> 이 문서는 프로덕션 환경에 필요한 GitHub Secrets 목록을 정의합니다.
> 실제 값은 기록하지 않으며, 어떤 종류의 시크릿이 필요한지만 명시합니다.

-   `VERCEL_TOKEN`: Vercel 배포를 위한 토큰
-   `VERCEL_ORG_ID`: Vercel 팀/조직 ID
-   `VERCEL_PROJECT_ID_PRODUCTION`: Vercel의 프로덕션 프로젝트 ID
-   `SUPABASE_URL_PROD`: 운영용 Supabase 프로젝트 URL
-   `SUPABASE_ANON_KEY_PROD`: 운영용 Supabase 익명 키
-   `OPENAI_API_KEY_PROD`: 운영용 OpenAI API 키
